# pbflags v0.21.0

This release introduces new scaffolding commands for faster flag development and enhances debugging capabilities.

## CLI & Development Tools

- Added `pb flag new <feature> <flag>` command to automatically scaffold new flag proto fields
- Added `pb feature new` and `pb config new` commands for creating boilerplate feature and configuration structures
- Reorganized flat configuration commands under improved command hierarchy

## Server

- Added configurable debug logging for flag evaluation to help troubleshoot flag resolution issues
- Enhanced launch page with targeted ramp swapping functionality and improved ramp display
- Renamed "Features" section for better clarity in the management interface
- Development database now runs on port 9202 for better separation from production setups
