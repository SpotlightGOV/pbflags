# pbflags v0.24.0

This release improves the reliability of condition evaluation and code generation with important bug fixes and internal architecture improvements.

## Server

- Fixed condition evaluation integration to properly wire the ConditionEvaluator into both the DBFetcher and main Evaluator components.

## Code Generation

- Fixed a nil pointer dereference issue in the protoc-gen-pbflags plugin that could cause crashes during code generation.
- Added upgrade warnings to help users identify when updates are needed.
