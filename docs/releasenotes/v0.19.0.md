# pbflags v0.19.0

This release focuses on improving testing capabilities, performance optimization, and operational monitoring.

## Testing

- Added `WithOverrides` option for test-only flag value overrides, enabling easier unit testing of flag-dependent code.
- Introduced comprehensive evaluator distribution tests and benchmarks with baseline performance data.

## Server

- Implemented bundle file watching with automatic conditions re-sync in standalone admin mode for real-time configuration updates.
- Optimized evaluation performance by removing per-evaluation spans and pre-currying metrics calculations.
- Added head sampling support to reduce telemetry overhead while maintaining observability.
