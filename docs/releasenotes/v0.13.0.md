# pbflags v0.13.0

This release focuses on improving system reliability and documentation for production deployments.

## Server

- Fixed schema migration issues in spotlight integration that could cause database sync failures.
- Improved error logging throughout the server to provide clearer diagnostic information.
- Added support for flexible URL formats with automatic normalization to handle various endpoint configurations.

## Documentation

- Enhanced agent setup guide with detailed error logging configuration examples.
- Added documentation for supported URL formats and connection troubleshooting.
