# pbflags v0.12.0

This release focuses on infrastructure improvements and dependency management enhancements.

## Infrastructure

- Updated Spotless plugin to version 8 for improved code formatting and style enforcement.
- Introduced pbflags-specific database migration versioning to prevent conflicts with other applications using Goose migrations.

## Migration Steps

- If you're running a pbflags server with existing database migrations, the migration version table will now use a pbflags-specific name to avoid conflicts with other Goose-managed schemas in the same database.
